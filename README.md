# Amélioration de Code Hérité

## Description
Cet outil est conçu pour aider les développeurs à interagir avec du code hérité de manière plus efficace. Il propose des fonctionnalités d'analyse, de refactoring automatique et des recommandations de tests, facilitant ainsi la maintenance et l'évolution des applications anciennes.

## Fonctionnalités
- **Analyse Statique**: Identifie les parties du code problématiques et propose des améliorations.
- **Refactoring Automatique**: Processeurs puissants permettant de restructurer le code sans changer son comportement.
- **Scénarios de Tests**: Génération automatique de scénarios de tests pour garantir que les modifications n'introduisent pas de bogues.
- **Interface Utilisateur Intuitive**: Simple à utiliser, même pour les développeurs moins expérimentés.

## Installation
Pour installer ce projet, vous pouvez cloner ce dépôt et installer les dépendances nécessaires :
```bash
git clone https://github.com/votre-utilisateur/amelioration-de-code-legacy.git
cd amelioration-de-code-legacy
npm install
```

## Utilisation
Une fois installé, vous pouvez lancer l'outil en utilisant la commande suivante :
```bash
npm start
```
